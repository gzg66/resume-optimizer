# 简历优化器使用说明（基于LLM + 自动读取 resume.txt）

## 核心思路

本技能**自动读取 resume.txt 文件**中的实习经历和项目经历，然后利用LLM的强大理解能力来完成简历优化！所有JSON字段名都是中文，更易读！

## 文件结构

```
简历-JD优化/
├── resume.txt                          # 实习经历和项目经历（固定读取）
└── .trae/skills/resume-optimizer/
    ├── basic_info.json                 # 基本信息（姓名、联系方式等）
    ├── optimize_resume.py              # 主脚本
    └── ...
```

## 快速开始

### 方法一：在Trae中直接使用（推荐）

当你需要优化简历时，直接告诉我：

> "帮我优化简历，岗位JD是..."

我会自动从 resume.txt 读取经历，调用LLM帮你完成优化！

### 方法二：使用脚本生成提示词

```bash
cd .trae/skills/resume-optimizer
python optimize_resume.py --init  # 创建示例基本信息
python optimize_resume.py --jd "岗位JD内容"
```

然后复制生成的提示词到你的LLM中即可。

## 基本信息格式（basic_info.json）

```json
{
  "姓名": "张三",
  "联系方式": {
    "邮箱": "zhangsan@example.com",
    "电话": "138-0000-0000"
  },
  "个人介绍": "热爱技术，具有3年软件开发经验，善于团队协作。",
  "技能": ["技能1", "技能2"]
}
```

## resume.txt 格式

resume.txt 包含你的实习经历和项目经历，格式可以是自由的markdown格式，示例：

```
实习1：
**项目：基于多智能体协同与混合架构的游戏自动化测试系统**

**【项目概述】**
...

项目1：
**项目：NPC 智能 Agent**
...
```

## LLM优化流程

1. **深度理解JD**：LLM分析岗位的核心要求
2. **智能筛选经历**：从 resume.txt 中挑选最匹配的工作/项目经历
3. **优化技能列表**：根据经历内容提取并优化技能
4. **润色个人介绍**：突出与JD匹配的能力和经验
5. **输出完美简历**：生成专业的优化结果

## 使用示例

### 对话示例

**你**：我要优化简历，应聘Python后端开发工程师。岗位JD是...

**我**：好的！我来帮你优化简历...（自动从 resume.txt 读取经历）

### 脚本示例

```bash
# 创建示例基本信息
python optimize_resume.py --init

# 生成优化提示词（使用默认的 basic_info.json）
python optimize_resume.py --jd "任职要求：熟练掌握Python，熟悉Django框架，有MySQL数据库经验，了解Docker。"

# 指定其他基本信息文件
python optimize_resume.py --jd "岗位JD" --basic my_info.json
```
