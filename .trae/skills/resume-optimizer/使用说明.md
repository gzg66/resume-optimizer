# 简历优化器使用说明（基于LLM）

## 核心思路

本技能不再使用脚本解析，而是**直接利用LLM的强大理解能力**来完成简历优化！

## 快速开始

### 方法一：在Trae中直接使用（推荐）

当你需要优化简历时，直接告诉我：

> "帮我优化简历，我的简历是...，岗位JD是..."

我会自动调用LLM帮你完成优化！

### 方法二：使用脚本生成提示词

```bash
cd .trae/skills/resume-optimizer
python optimize_resume.py --init  # 创建示例简历
python optimize_resume.py --jd "岗位JD内容" --resume sample_resume.json
```

然后复制生成的提示词到你的LLM中即可。

## 简历数据格式

```json
{
  "name": "姓名",
  "contact": {
    "email": "邮箱",
    "phone": "电话"
  },
  "introduction": "个人介绍",
  "skills": ["技能1", "技能2"],
  "experiences": [
    {
      "company": "公司",
      "position": "职位",
      "period": "时间",
      "description": "描述",
      "achievements": "成就"
    }
  ],
  "projects": [
    {
      "name": "项目名称",
      "description": "项目描述"
    }
  ]
}
```

## LLM优化流程

1. **深度理解JD**：LLM分析岗位的核心要求
2. **智能筛选经历**：挑选最匹配的工作/项目经历
3. **优化技能列表**：优先展示与JD匹配的技能
4. **润色个人介绍**：突出匹配的能力和经验
5. **输出完美简历**：生成专业的优化结果

## 使用示例

### 对话示例

**你**：我要优化简历，应聘Python后端开发工程师。这是我的简历...，这是岗位JD...

**我**：好的！我来帮你优化简历...（自动调用LLM）

### 脚本示例

```bash
# 创建示例简历
python optimize_resume.py --init

# 生成优化提示词
python optimize_resume.py --jd "任职要求：熟练掌握Python，熟悉Django框架，有MySQL数据库经验，了解Docker。" --resume sample_resume.json
```

## 优势

✅ **更智能**：LLM深度理解语义，不只是关键词匹配  
✅ **更专业**：专业的简历优化专家提示词  
✅ **更灵活**：支持各种简历格式和JD内容  
✅ **更高效**：一键生成，直接使用
